# Zerobus Flow
> Streaming data is essential to our data ecosystems like water is to the human body. Without it, we fail to exist.

This project aims to make it incredibly simple to utilize the `Zerobus` SDKs - including providing documentation 
on what works, what gotchas could get in the way of your immediate success, and lastly aims to utilize
[protobuf](https://protobuf.dev/) whereever possible.
